CPU Scheduling Simulator

What is CPU Scheduling?

CPU Scheduling is a process used by the Operating System to decide which process gets the CPU next. Since only one process can use the CPU at a time, scheduling algorithms ensure fairness and efficiency.

How to Use

  • Enter Arrival Time and Burst Time for each process.
  • Select a Scheduling Algorithm.
  • If using Round Robin, enter Time Quantum.
  • If using Priority, set priority as Lower=Higher or Higher=Higher.
    • Lower=Higher represents smaller number → higher priority
    • Higher=Higher represents larger number → higher priority
  • Optional: Hybrid scheduling (multiple algorithms in one simulation) is not active by default. It will be available if the user selects the "Hybrid Mode" option.
  • Click Run Simulation.
