What is Memory Optimization?

The OS manages how memory (RAM) is allocated and freed for processes.

How to Use ?

  4. Click Start Simulation.

Key Terms Explained

  • Page: A block of memory referenced by the CPU.
  • Frame: A fixed-size slot in physical memory that holds a page.
  • Page Fault: Occurs when the required page is not found in memory.
  • Hit: Occurs when the required page is already present in memory.
  • Page Fault Rate (PFR): Faults ÷ Total References.
  • Reference String: Sequence of page requests made by CPU.
