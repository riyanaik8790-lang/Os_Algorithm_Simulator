Disk Scheduling Simulator

What is Disk Scheduling?

Disk Scheduling is an Operating System mechanism used to decide the order in which pending disk I/O requests are serviced. Since the time required to access data on a disk largely depends on the movement of the disk head, efficient scheduling is essential to reduce seek time, minimize latency, and improve overall system performance. Different disk scheduling algorithms apply different strategies to optimize head movement, ensure fairness, and maximize throughput.

How to Use ?

  1. Enter the disk request sequence (comma separated values).
  2. Enter the initial head position.
  3. Select the scheduling algorithm (FCFS, SSTF, SCAN, etc.).
