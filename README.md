# 💻 OS Algorithm Simulator

An interactive web-based simulator for Operating System algorithms with graphical visualization and step-by-step calculations.

This project demonstrates core OS concepts like CPU Scheduling, Disk Scheduling, and Page Replacement in an easy-to-understand and visual way.

---

## 🚀 Features

### 🖥 CPU Scheduling Algorithms
- FCFS (First Come First Serve)
- SJF (Shortest Job First)
- SRTN (Shortest Remaining Time Next)
- Priority (Preemptive & Non-Preemptive)
- Round Robin
- Gantt Chart Visualization
- Waiting Time & Turnaround Time Calculation
- Hybrid Scheduling

### 💾 Disk Scheduling Algorithms
- FCFS
- SSTF
- SCAN
- C-SCAN
- Graphical Head Movement Display
- Total Head Movement Calculation
- Average Seek Time Calculation

### 📄 Page Replacement Algorithms
- FIFO
- LRU
- Optimal
- Page Fault Calculation
- Page Fault Ratio (PFR)
